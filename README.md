# uno-test

Тестовое задание для uno-soft

## Описание

- Использована **Java 24.0.2**
- Сборщик: **Gradle 9.4.1**


## Алгоритм

Программа считывает строки из входного файла (в том числе `.gz`) и группирует их по совпадению значений в одинаковых 
колонках. Если строки имеют общее значение в одной колонке, они попадают в одну группу, при этом связанные группы 
автоматически объединяются. В результате формируется список групп, отсортированных по размеру, и выводится количество 
групп, содержащих более одного элемента.

## Запуск

Для запуска используйте команду:

```bash
java -Xmx1G -jar {название проекта}.jar {полный путь к входному файлу}
```
